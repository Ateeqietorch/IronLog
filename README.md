# IRONLOG — Hypertrophy Tracker

A standalone workout tracking web app that saves permanently to Google Sheets.

## How to Deploy (5 minutes)

### Step 1 — Create a GitHub account
If you don't have one, go to github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it: `ironlog`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. On your new repo page, click **Add file → Upload files**
2. Upload all three files:
   - `index.html`
   - `style.css`
   - `app.js`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo **Settings**
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch, **/ (root)** folder
5. Click **Save**

### Step 5 — Open your app
After about 60 seconds, your app will be live at:
`https://YOUR-GITHUB-USERNAME.github.io/ironlog`

Bookmark this on your phone's home screen and it works like a native app.

## Features
- Log sets, reps, and weight for each exercise
- See your last session for each training day
- Auto-progression: weights bump up/down based on performance
- Calendar view of all logged sessions
- Progress history showing weight changes over time
- All data saved permanently to your Google Sheet
